# Klasifikasi Penyakit Gugur Daun Karet Menggunakan SE-ResNet-50

Repository ini berisi kode sumber penelitian skripsi mengenai
klasifikasi tingkat keparahan penyakit gugur daun pada daun karet
menggunakan arsitektur SE-ResNet-50.

## Dataset
Dataset tidak disertakan dalam repository ini.
Struktur dataset yang digunakan:
dataset/
├── 0/
├── 1/
├── 2/
├── 3/
└── 4/

## Model
Model yang digunakan adalah SE-ResNet-50 dengan mekanisme
Squeeze-and-Excitation.

## Deployment
Deployment dilakukan secara sederhana dengan menggunakan
model terlatih untuk melakukan prediksi pada citra uji.
